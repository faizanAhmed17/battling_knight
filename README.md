# Challenge: Battling Knights

Steps:
- Create an input file for moves with a name "moves.txt", having the following format
```
    GAME-START
    R:S
    R:S
    B:E
    G:N
    Y:N
    GAME-END
```
- execute ``` python main.py```
- Final state can be viewed in final_state.json 


*Note*: This is python3 code
